# TeamSpirit Info Display

TeamSpiritのホーム画面に勤怠情報を直接表示するChrome拡張機能です。

## 機能

打刻エリアの横に以下の情報を表示します：

- **状態表示**: 出勤中 / 退勤済み / 未出勤
- **時間表示**:
  - 出勤時刻
  - 退勤時刻（退勤済みの場合）
  - 勤務時間（リアルタイム更新）
  - 目標退勤時刻（出勤中の場合）
- **月間サマリー（折りたたみ式）**:
  - 所定労働時間
  - 総労働時間
  - 過不足時間
  - 残り勤務日数
  - 一日当たり必要勤務時間

## スクリーンショット

打刻エリアの右側に情報パネルが表示されます：

```
┌─────────────────┐  ┌──────────────────────┐
│   打刻エリア     │  │    勤怠情報          │
│                 │  │                      │
│  2026/01/15     │  │  状態: 出勤中        │
│     8:21        │  │  出勤時刻: 07:53     │
│                 │  │  勤務時間: 01:30:00  │
│ [出勤] [退勤]   │  │  目標退勤: 17:00     │
│                 │  │                      │
└─────────────────┘  │  ▼ 月間サマリー      │
                     │    所定: 160:00      │
                     │    総労働: 80:00     │
                     │    過不足: -80:00    │
                     └──────────────────────┘
```

## インストール方法

1. このリポジトリをクローンまたはダウンロード
2. Chromeで `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」をオンにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. ダウンロードしたフォルダを選択

## ファイル構成

```
teamspirit-info-display/
├── manifest.json    # 拡張機能の設定
├── content.js       # コンテンツスクリプト（ページに注入）
├── styles.css       # 表示スタイル
├── README.md        # このファイル
└── icons/           # アイコン画像
    ├── icon16.png
    ├── icon48.png
    └── icon128.png
```

## 特徴

- **ページ内表示**: ポップアップを開かずにTeamSpiritページ上で直接確認
- **リアルタイム更新**: 勤務時間が1秒ごとに自動更新
- **自動検出**: TeamSpiritのホームページを開くと自動的に表示
- **折りたたみ可能**: 月間サマリーは折りたたんで省スペース化可能

## 注意事項

- TeamSpiritのホーム画面でのみ動作します
- ページ構造が変更された場合、表示されない可能性があります
